 ## Quote Generator Web App

##  Features

-  Generate quotes by topic
-  Random quote generator
-  Styled using ShadCN UI + Tailwind CSS
-  Built with Next.js App Router

---

##  Supported Topics

- Motivation
- Inspiration
- Success
- Happiness
- Friendship

> More topics can be easily added from the local quotes JSON.
